# 蓝莓软件官网

蓝莓软件组织和蓝莓软件的文档

> 给用户极致的软件使用体验，让工具回归工具，让生活回归生活

## 文档地址

* Netlify: <https://sky-app-documents.netlify.app/>

## 安装教程

1. 运行 `yarn install` 安装依赖
2. 运行 `yarn dev` 开发文档
3. 运行 `yarn build` 构建文档
